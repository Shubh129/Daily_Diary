CSS Syntax:

selector {
  property: value;
}

-Selector → Specifies which HTML element(s) to style.
-Property → The style attribute you want to change (like color, font-size, etc.).
-Value → The setting you want to apply to the property.

CSS Selectors:
-Selectors are used to "select" the HTML elements you want to style.

1. Universal Selector
Definition: Selects all elements on the page.
-Syntax:
* {
  margin: 0;
  padding: 0;
}

Use: Often used for resetting default browser styles.

2. Element Selector
Definition: Targets specific HTML tags (elements).
-Syntax:
h1 {
  color: blue;
}

Use: Applies style to all <h1> elements.

3. Class Selector
Definition: Targets elements with a specific class attribute.
-Syntax:
.myClass {
  font-size: 18px;
}

Use in HTML:
<p class="myClass">Styled paragraph</p>

4. ID Selector
Definition: Targets a specific element with a unique id.
-Syntax:
#mainHeading {
  color: red;
}

Use in HTML:
<h1 id="mainHeading">Welcome</h1>

5. Group Selector
Definition: Targets multiple selectors at once.
-Syntax:
h1, h2, p {
  font-family: Arial;
}

Use: Applies same style to different elements without repeating.

6. Pseudo-class Selector
Definition: Applies styles when an element is in a specific state or condition.
-Syntax:
selector:pseudo-class {
  property: value;
}

-Common Pseudo-classes:
:hover
Style when mouse is over an element.
a:hover {
  color: green;
}

:focus
Style when an element (like input) is focused.
input:focus {
  border: 2px solid blue;
}

:first-child
Selects the first child of a parent.
p:first-child {
  font-weight: bold;
}

:last-child
Selects the last child of a parent.
li:last-child {
  color: red;
}

Table
| Selector Type | Symbol | Example      | Target                      |
| ------------- | ------ | ------------ | --------------------------- |
| Universal     | `*`    | `* {}`       | All elements                |
| Element       | —      | `p {}`       | All `<p>` tags              |
| Class         | `.`    | `.box {}`    | Elements with `class="box"` |
| ID            | `#`    | `#header {}` | Element with `id="header"`  |
| Group         | `,`    | `h1, p {}`   | All `<h1>` and `<p>` tags   |
| Pseudo-class  | `:`    | `a:hover {}` | `<a>` tags when hovered     |
